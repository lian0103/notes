# GT前端專案模板

版本號: v0.0.10-beta-4
整理常用的工具/庫，包含Vite、TailwindCSS、SCSS、Auto Import等配置。在概念上把他打包成一個NPM的npx腳手架工具，把模板的結構做COPY後來建立新專案。

# 建立專案

npx gt-template@[版本號] [projectName] [template]

// ex. npx gt-template Hello 

// ex. npx gt-template Hello elementPlus
